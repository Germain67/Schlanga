Comment l'installer?
Lancer le script install.sh pour installer les librairies SDL. 
Ensuite, il suffit de faire make pour compiler le jeu puis ./bin/main pour lancer le jeu.


Comment jouer?
Le déplacement dans le menu se fait grâce à l'aide des touches directionnelles HAUT et BAS. Pour choisir un menu, il suffit ensuite d'appuyer sur la touche Entrer.
Les menus Scores et Options ne sont pas encore fonctionnels. Ils seront développés ultérieurement.
Afin de jouer au jeu, il suffit d'aller dans le menu Jeu. Le snake, de couleur verte, est jouable grâce aux touches directionnelles.
